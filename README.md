## ▶️ Running the Programs

Clone the repository:

```bash
git clone https://github.com/Nipun-Lakshan/python-course-jenny.git
```

Navigate to the project directory:

```bash
cd python-course-jenny
```

Run a Python file:

```bash
python filename.py
```
